# Local_Network_Drive

**A clean, modern, self-hosted personal file manager & drive**  
Simple. Beautiful. Mobile-friendly.

Pouya Drive is a lightweight, private file storage and sharing solution you can run on your own server (or even locally).

### Features
- Clean dark-themed interface with glassmorphism / neumorphism style
- Folder navigation with breadcrumb-style back button
- Upload files (single + multiple supported)
- Create new folders
- Download files with real-time progress
- Delete files & folders (with confirmation)
- Progress overlay with speed & percentage during upload/download
- Fully responsive design — works great on phones & tablets
- Path-safe navigation (prevents directory traversal attacks)
- Built with Flask (Python) + vanilla JavaScript + modern CSS
